# E: Projects

Consolidated Python projects following modern src layout best practices.

## Projects

- **GRID** - Geometric Resonance Intelligence Driver
- **Coinbase** - Crypto-focused agentic system
- **wellness_studio** - Medical wellness AI pipeline
- **utilities** - Shared utility scripts and tools

## Structure

All projects use the src layout:
```
project_name/
├── src/package_name/       # Importable package code
├── tests/                  # Test files
├── scripts/                # Utility scripts
├── docs/                   # Documentation
├── pyproject.toml          # Package configuration
└── README.md
```
